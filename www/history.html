Welcome to HTML's HISTORY!


This is the history section. This section will briefly discuss the history of HTML including the man behind this language and how and where it started. You'll also see here the transition of versions of HTML.

It All Started Here.


Tim Berners-Lee, a British computer scientist, and the inventor of the World Wide Web or most commonly known as the "Web". It all started at CERN, a large physics laboratory in Geneva, Switzerland where Tim was working that time. Tim came up with an idea that will let researchers around the world to put together information in an organize way. This, according to Tim, can be done using "links". This idea will also enable researchers to download files and save it to individual computers. To sum it up, all these things can be done using hypertext which is a way of linking documents together by clicking buttons on the screen. Tim already had develop this hypertext system in the year 1980 before he worked for CERN, it is called Enquire.

The Evolution


HTML1.0


The first version publicly available of HTML was published late 1991 and was a document called "HTML tags". It has 18 tags available for use and more than half of that 18 tags are still in use in the curent version of HTML (4.01)

RFC 1866


HTML2.0 or RFC 1866. It includes everything from the original version (1.0). It was the standard webdesign until 1997.

HTML3.0


Dave Raggett introduced a new draft called HTML 3.0. This version was a failure because browsers took large amount of time implementing its features.

W3C and WILBUR


W3C(World Wide Web Consortium) was founded in 1994 and was responsible for creating the HTML3.2, code-named "Wilbur". It became the official webdesign standard in 1997 and up to know, all browsers still support its features.

Codename "COUGAR"


Initially called "Cougar", HTML4.0 was recommended by W3C and later became the official standard of webdesign in Aprill 1998. It includes CSS for an easier way to modify and improve the looks of the HTML file.

The Final Revision


HTML4.01 was the final revision of HTML and the current version being use at the present time. XHTML was also introduce this time. It's a combination of XML and HTML.

HTML5


HTML5 surfaced at the time when XHTML2.0 was being proposed. Since XHTML2.0 is starting to look bad, a group of developers outside W3C started to build something on their own. Web Hypertext Application Technology Working Group(WHATWG) was the name of the group responsible for HTML5. Although it's still under development, some features of HTML5 are already supported by some browsers.
